# AWS EC2 and Nginx Web Server Project (CLI only)

This project demostrates the the use of command-line interface to launch an EC2 instance.
Also to install an Nginx server into the EC2 instance and enable the server.

# The Services Used for this Project
- AWS IAM
- AWS EC2 Instance
- Nginx
- AWS CLI
- Git & GitHub

# Major Activities of this Project
- Installed an AWS cli
- Configured the AWS cli for the project
- Launched an EC2 Instance
  - Created a key-pair
  - Create a security group 
  - Described the amazon machine image (AMI)
- Connected to the EC2 server via command-line secure shell (SSH)
- Installed Nginx into the EC2 instance
  - started Nginx
  - Enabled Nginx
  - Checked its functionality (running)
- Hosted an HTML web
- Checked the functionality of Nginx with HTML web
